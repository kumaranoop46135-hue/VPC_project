A **Virtual Private Cloud (VPC)** is a private network inside AWS.  
It is like our own personal network where we can securely deploy cloud resources.

Inside a VPC we can create:

- Public Subnet  
- Private Subnet  
- Route Tables  
- Internet Gateway  
- Security Groups  
- Load Balancer
